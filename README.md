# iSH64 – 64-bit Linux Environment for iOS

**Author:** Victor Jose Corral  
**Project:** ish64-ios-port  
**Repository:** https://github.com/GeneXisD/ish64-ios-port

---

## Overview

iSH64 is an experimental extension of the iSH Linux emulator designed to explore
64-bit Linux userland environments on iOS devices. The project investigates
methods for supporting modern Linux toolchains and development workflows within
the constrained mobile runtime environment.

The goal of this project is to create a portable Linux development and security
research environment capable of running directly on iOS hardware.

---

## Key Features

- 64-bit Linux userland experimentation
- Custom build environment for Linux toolchains
- Integration with GitHub CI runners
- Networking experimentation environments
- Security research sandbox capabilities

## Technical Focus
